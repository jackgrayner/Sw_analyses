# Sw_analyses
Analyses of Sw mutants for manuscript 'Novel adaptation escapes evolutionary loss by recruiting life-history trade-offs'

## GWAS
Scripts used to identify genetic variants associated with Sw expression.

Genome sequencing data available from NCBI SRA: PRJNA952183 (newly uploaded) and PRJNA948842 (https://doi.org/10.1101/2025.08.30.673289). Reads were aligned to the T. oceanicus genome v3 (ENA accession GCA_976985615) 

## temp_winglen
Script and data file used to assess relationship between rearing temperature and Sw expression
